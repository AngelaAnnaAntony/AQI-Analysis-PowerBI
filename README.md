# AQI Quality Analysis Dashboard

##  Project Overview
This project analyzes air quality trends across multiple cities using Power BI. The dashboard provides insights into pollution levels, trends over time, and regional comparisons.

##  Key Features
- KPI metrics for major pollutants (SO2, CO, NO)
- Identification of top polluted and least polluted cities
- AQI trend analysis over time
- Map-based visualization for area-wise AQI distribution
- Interactive filters for city selection

##  Key Insights
- Certain cities consistently show higher AQI levels indicating severe pollution
- Noticeable variation in pollution levels across regions
- AQI trends highlight potential improvement or decline over time

## Tools Used 
- Power BI  
- Data Visualization
